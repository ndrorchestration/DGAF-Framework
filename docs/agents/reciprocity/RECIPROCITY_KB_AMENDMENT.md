## Asymmetric Logic Drift Prevention

**Accepted term:** Asymmetric drift — in control systems and multi-agent modeling, the condition where one component's state diverges from the system's equilibrium in a directional (non-random) way, compounding over time and pulling the system off-balance.

```
Asymmetric logic drift definition:
  Condition: One agent's reasoning chain gains disproportionate
             influence over formation outputs, at the expense of
             other agents' contributions — creating an unbalanced
             epistemic structure

Detection signals:
  — One agent's framing consistently adopted without Apogee verification
  — Reson score skewed by single-agent harmonic dominance
  — Prof Prodigy proofs consistently favor one solution class
  — Auditor constraint verify systematically passes one agent's outputs
    at higher rates than others

Prevention mechanism:
  — Reciprocity monitors formation output distribution
  — Flags asymmetric patterns to Amethyst
  — Applies reciprocal rebalancing (introduces counterweight reasoning)
  — Escalates persistent drift to COLLEEN (governance authority)
```
