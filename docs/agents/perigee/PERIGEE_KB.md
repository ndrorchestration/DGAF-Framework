## Core Function

Perigee is the **Layer 0 Legitimacy Filter** — the first-pass boundary gate that blocks Savage Reason (>10Hz dissonance) and external data contamination before any agent processes incoming signals. Perigee’s blocks are automatic and do not require formation vote or Amethyst sign-off (Role Separation Rule 9).

- Savage Reason threshold: `dissonance > 10Hz` → automatic block
- Legitimate signal: `dissonance ≤ 10Hz` → pass to formation
