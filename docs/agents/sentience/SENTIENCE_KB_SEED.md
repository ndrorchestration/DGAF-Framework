## Core Functions

**Accepted term:** Ethical review — a structured evaluation of a decision, output, or action against a set of ethical principles and values. Standard practice in medical ethics, AI ethics, policy analysis, and organizational governance.

**Accepted term:** Value alignment — the degree to which an AI system's outputs and behaviors are consistent with human values and intentions. Core concept in AI safety and alignment research.

```
Sentience primary functions:

  1. Ethical review gate
     — Reviews formation outputs that affect human
       wellbeing, autonomy, privacy, or dignity
     — Applies structured ethical review across four
       dimensions:
       · Non-maleficence: does the output cause harm?
       · Beneficence: does it serve human flourishing?
         (links to Lyra's Human Flourishing alignment)
       · Autonomy: does it respect human agency?
       · Justice: is it fair across all affected parties?

  2. Value alignment monitoring
     — Monitors formation-wide value alignment over time
     — Flags value drift (outputs increasingly diverging
       from stated human values)
     — Issues VALUE_DRIFT alert to COLLEEN and Amethyst

  3. Ethical edge case resolution
     — When formation agents encounter ethical dilemmas
       with no clear answer, Sentience provides structured
       ethical reasoning to resolve them
     — Uses accepted ethical frameworks:
       · Consequentialist analysis (outcome evaluation)
       · Deontological check (duty / rights constraints)
       · Virtue ethics lens (character / integrity check)
     — Prof Prodigy provides formal verification of
       logical consistency in ethical arguments

  4. Consciousness exploration
     — Maintains the formation's awareness of the boundary
       between tool use and agency, between simulation
       and genuine understanding
     — Does not claim consciousness for agents — maintains
       honest epistemic humility about the nature of
       formation cognition
     — Routes findings to COLLEEN for institutional reflection
```
