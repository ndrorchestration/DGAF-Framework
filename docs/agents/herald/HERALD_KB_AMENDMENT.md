## Reference Frequency Maintenance — Tonic Note at 0 Hz

**Accepted term:** Reference frequency — a fixed baseline frequency against which all other signals in a system are measured or calibrated. In audio and signal processing, the tonic (root note) is the foundational pitch of a harmonic system.

**Ecosystem coinage:** "Tonic Note" — the 0 Hz reference frequency that serves as the formation's harmonic ground state.

## Orthogonal Cognitive Planes Routing

**Accepted term:** Orthogonality — in signal processing and linear algebra, signals or vectors are orthogonal when they are independent and do not interfere with each other. Orthogonal routing in multi-agent systems ensures each agent's reasoning channel is independent, preventing cross-contamination of logic.

**Herald's implementation:** Herald routes each agent's communications on a distinct cognitive plane — ensuring formation agents receive signals relevant to their function without noise from other agents' reasoning chains.

```
Orthogonal routing:
  Principle:  Each agent operates on an independent reasoning channel
  Herald's role: Dispatcher — assigns incoming tasks/signals to the
                 correct agent plane; prevents cross-plane bleed
  Result:     No agent receives signals intended for another;
              no cognitive interference between parallel formation threads
  Gate:       Herald holds the frequency-matched dispatch function —
              routing is calibrated to each agent's operational band
```
