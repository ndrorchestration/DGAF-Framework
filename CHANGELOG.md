## [1.0.0] - 2026-01-04

### License Change

**BREAKING CHANGE: License migrated from MIT to Apache 2.0**

#### Rationale

This project has transitioned to Apache License 2.0 to provide:

1. **Explicit patent grants** protecting institutional users from patent litigation
2. **Clear modification documentation requirements** supporting security governance
3. **Enterprise-grade legal clarity** for formal procurement and compliance processes

As a security framework designed for institutional and enterprise deployment, Apache 2.0 ensures explicit patent protection critical for security implementations.

#### Compatibility

- **Past releases**: Remain available under MIT License terms
- **Current release** (v1.0.0+): Apache License 2.0 only
- **No action required** for existing MIT-licensed deployments
- **New deployments** should review Apache 2.0 requirements (see LICENSE and NOTICE files)

#### Impact on Users

- **No breaking changes** to code functionality
- **Same permissive terms** (commercial use, modification, distribution allowed)
- **Added protections** via explicit patent grants and termination clauses
- **Additional requirement**: Include LICENSE and NOTICE files in distributions

For security-conscious and institutional users: This change provides enhanced legal protection aligned with enterprise security governance practices.
